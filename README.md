CyberSafe AI – Cybersecurity Awareness Platform

Problem Statement

ACS-3: Cybersecurity Awareness & Education Platform
Many internet users fall victim to phishing messages, fake websites, and online scams due to lack of cybersecurity awareness. Our project aims to help users detect suspicious messages and websites while educating them about safe online practices.

Project Description

CyberSafe AI is a simple web-based platform that helps users identify potential scam messages and suspicious websites. The platform analyzes message content and URL patterns to warn users about possible phishing attempts. It also provides cybersecurity tips to educate users on how to stay safe online.

Features

Scam Message Detection

Suspicious Website URL Checker

Cybersecurity Awareness Tips

Simple and user-friendly interface

How It Works

The user pastes a message into the scam detection tool.

The system checks for suspicious keywords such as OTP, urgent, click, bank.

If risky patterns are found, the system warns the user that the message may be a scam.

Users can also enter a website URL to check if it looks suspicious.

The platform also displays cybersecurity safety tips to educate users.
